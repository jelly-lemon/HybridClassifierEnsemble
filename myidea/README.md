# 自适应采样率思想
问：为什么要设计采样间隔呢？   
答：用到了 Bagging 思想，采样率不同，得到的数据集差别较大，训练结果也较大，用于粒子群算法，增加样本多样性。    


## 采样间隔设计原则   
1、从不采样到平衡采样率   
不采样：那就是采样率为 1 的意思，取所有数据。      
平衡采样率：多数类经过采样后数量刚好与少数类样本数量相同。

2、采样间隔不能 0< 或 >1
