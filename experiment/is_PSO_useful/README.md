# 实验目的
PSO 粒子群优化是否有效果？做个实验看一下。

# 实验内容
粒子群优化算法，只使用一个目标函数，那就是 min|s*|ui-uj|^2|。

## 目标函数思想
该目标函数是一个启发式函数。
基于这样的启发：两个样本越相似，那么同属于一个类别的概率也越大。
所以我们认为，所有样本的 “样本相似度 * 样本类别差距” 加起来，值越小，预测结果越准确。
